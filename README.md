# 文件快传 - Quicker

一个简单易用的局域网文件和文本传输工具，基于Flask开发，支持多设备间快速传输文件和文本内容。

## 功能特性

- 📁 **文件传输**
  - 支持拖拽上传文件
  - 支持多文件同时上传
  - 显示上传进度
  - 文件预览和管理
  - 单个文件下载
  - 批量下载（ZIP格式）

- 📝 **文本传输**
  - 发送文本内容
  - 文本历史记录
  - 一键复制文本内容

- 📱 **便捷访问**
  - 生成二维码，手机扫码访问
  - 自动检测本机IP地址（IPv4/IPv6）
  - 支持多个设备间互传

## 系统要求

- Python 3.7 或更高版本
- Flask 框架
- 支持现代浏览器的设备（用于访问Web界面）

## 安装与使用

### 方法一：使用批处理文件（Windows）

直接双击运行 `运行服务.bat` 文件，系统将自动启动服务。

### 方法二：命令行运行

1. 打开终端或命令提示符
2. 导航到项目根目录
3. 运行以下命令：

```bash
python main.py
```

### 首次运行

首次运行时，系统会自动安装所需依赖并创建上传目录。

## 使用说明

1. 启动服务后，控制台会显示可用的访问地址
2. 在浏览器中打开显示的地址（推荐使用第一个）
3. 可以通过二维码扫描在移动设备上访问
4. 在"发送文件"标签页可以上传和下载文件
5. 在"发送文本"标签页可以发送和接收文本内容
